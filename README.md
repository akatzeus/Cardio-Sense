# CardioSense

CardioSense is a full-stack web application designed to monitor heart health using
IoT sensor data and machine learning. The system focuses on early anomaly detection
and predictive insights to help identify potential cardiovascular risks.

---

## Tech Stack

### Frontend
- React (Vite)
- TypeScript
- Tailwind CSS
- shadcn/ui

### Backend
- Flask (Python)
- RESTful APIs
- Flask-CORS
